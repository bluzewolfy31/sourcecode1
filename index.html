What is Coding?

Coding itu Bahasa Yang dipakai untuk memprogram Komputer Coding itu bisa digunakan untuk Membuat aplikasi/Website,game,dan animasi.

Website

Website adalah kumpulan halaman Web yang dapat diakses melalui sebuah alamat menggunakan browser.

HTML adalah bahasa yang akan kita gunakan untuk membuat kerangka dari website.

CSS adalah bahasa yang kita gunakan untuk mempercantik website yang kita buat.

JavaScript adalah bahasa yang digunakan untuk membuat website menjadi lebih responsive.

What is Construct 3?

Construct 3 itu aplikasi yang membantu kamu untuk membikin game seperti game puzzle,shooting,maze,dll.
